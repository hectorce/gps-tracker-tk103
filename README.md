# gps-tracker-tk103

A receiver and web-ui to receive and display GPS data from TK103 based GPS trackers.

## Build

```
git clone git@github.com:roobert/gps-tracker-tk103
cd gps-tracker-tk103
make
```

## Run

Web UI/API
```
./gps-tracker-tk103-ui
```

Data receiver
```
./gps-tracker-tk103-receiver
```
